## Required Behavior

### Admin Authorization

Every admin route must verify:

```ts

```

#### Backend

Immediately:

1. Invalidate the JWT session.
2. Clear all authentication cookies.
3. Destroy any session data.
4. Return HTTP 401 Unauthorized.
5. Never redirect to the user dashboard.

Example:

```ts

    success: false,
    message: "Unauthorized",
    redirect: "/signin"
});
```

When receiving a 401 response:

1. Clear localStorage.
2. Clear sessionStorage.
3. Remove any cached user data.
4. Reset auth context/state.
5. Force logout.
6. Redirect to:

```ts
/signin
```

The user must not be sent to the dashboard.

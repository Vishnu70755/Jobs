# Google Calendar Integration

Requirements:

* Whenever a user creates or updates an interview date in the Tracker section, automatically create or update the corresponding Google Calendar event.
* Changes to interview date, time, company name, role, or notes should immediately synchronize with Google Calendar.
* Deleting an interview should remove the event from Google Calendar.
* Prevent duplicate calendar events.
* Store the Google Calendar event ID in the database.
* Show a confirmation message when synchronization succeeds.
* Handle failures gracefully with error notifications.
* Use Indian timezone (Asia/Kolkata).

The calendar event should contain:

* Company Name
* Job Role
* Interview Date and Time
* Interview Mode (Online/Offline)
* Location or Meeting Link
* Notes and Preparation Tips
